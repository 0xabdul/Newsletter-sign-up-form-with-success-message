# Frontend Mentor - Newsletter sign-up form with success message<br>

### Overview:<br>

- The project aims to create a simple yet effective newsletter sign-up form for a website. This form will allow visitors to input their email addresses and subscribe to a newsletter. Upon successful submission, a confirmation message will be displayed to the user using JavaScript.


- Sign-Up Form: The form will have fields for the user to input their email address.
  
  
- Validation: Basic client-side validation will be implemented to ensure that the email field is not empty and follows a valid email format.
  

- Success Message: Upon successful submission of the form, a success message will be displayed to the user.
  

- Responsive Design: The form will be designed to be responsive, ensuring it looks good and functions well on different devices and screen sizes.<br>
